用于认证登录，该系列接口不通过鉴权

## 获取验证码图片

<GET>/captcha</GET>  

### 请求

<TypeTable
  type={{
    w: {
      description: '图片宽度',
      type: 'string',
      default: 120,
    },
    h: {
      description: '图片高度',
      type: 'string',
      default: 50,
    },
    background: {
      description: '背景CSS',
      type: 'string',
    },
  }}
/>

## 判断是否需要提供验证码

<GET>/captcha/flag</GET>

### 请求

无参数

### 响应

<TypeTable
  noRequiredMark={true}
  type={{
    showCaptcha: {
      description: '是否展示验证码',
      type: 'boolean',
      required: true,
    },
  }}
/>

```json title="示例"
{
  "showCaptcha": false
}
```
