## 运行

<POST>/exec/file</POST>

### 请求

<TypeTable
  type={{
    path: {
      description: '代码文件路径',
      type: 'string',
      required: true,
    },
    options: {
      description: '命令选项配置',
      type: '{ key: string; value: string }[]',
      required: false,
    },
    envs: {
      description: '环境变量配置',
      type: '{ key: string; value: string }[]',
      required: false,
    },
  }}
/>

### 响应

返回事件 ID 字符串，关于如何获取日志详见 [获取实时日志](/docs/api/websocket/log)。


## 调试运行

<POST>/exec/file/debug</POST>

会创建临时文件并在运行完毕后自动清理，不修改原始代码文件内容。

### 请求

<TypeTable
  type={{
    path: {
      description: '代码文件路径',
      type: 'string',
      required: true,
    },
    content: {
      description: '代码文件内容',
      type: 'string',
      required: true,
    },
    options: {
      description: '命令选项配置',
      type: '{ key: string; value: string }[]',
      required: false,
    },
    envs: {
      description: '环境变量配置',
      type: '{ key: string; value: string }[]',
      required: false,
    },
  }}
/>

### 响应

返回事件 ID 字符串，关于如何获取日志详见 [获取实时日志](/docs/api/websocket/log)。
