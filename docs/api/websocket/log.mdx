---
title: 获取实时日志
---

持续推送日志数据，消息格式如下

<Callout icon={false}>
接口路径：`/api/ws`  
事件名称：`runLog`
</Callout>

<TypeTable
  noRequiredMark={true}
  type={{
    runId: {
      description: '事件 ID',
      type: 'string',
      required: true,
    },
    log: {
      description: '日志内容',
      type: 'string',
      required: true,
    },
    over: {
      description: '是否运行完毕',
      type: 'boolean',
      required: true,
    },
  }}
/>
